## 新拟态UI风格

&emsp;&emsp;新拟态设计风格是一种通过控制阴影的投射角度、深浅程度和模糊半径，来模拟真实物体形态的UI设计风格。逼真的立体感、简洁的科技感，使其在扁平化设计的主流中突出重围，成为 2021 年 UI 设计风格的一匹黑马。
